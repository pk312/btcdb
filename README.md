### DE:
## Bitcoin-only Dashboard
Dieses Dashboard stellt eine optimierte Übersicht aller relevanten Informationen rund um deinen Bitcoin-Bestand zur Verfügung. Es soll Neueinsteigern aber auch langjährigen Bitcoin-Investoren helfen, einen ausgewogenen Überblick der getätigten Transaktionen zu erhalten und mit aktuellen Marktwerten abzugleichen.
Es können eigene Transaktionen eingetragen werden (auch Transaktionen aus der Vergangenheit). Das Dashboard wertet diese Daten aus und visualisiert anhand der aktuellen Marktwerte die eigenen BTC-Bestände.


### Folgende Eigenschaften stehen zur Verfügung:
- automatisierte Marktwertabfragen
- Transaktionserfassung (auch in unterschiedlichen Währungen möglich)
- Wallet-Verwaltung und Übersicht
- Auswertungsdiagramme
- Solominer-Monitoring (AxeOS)
- Bericht erstellen (Kontoübersicht)
- 24-Stunden Preishistorie

### EN:
## Bitcoin-only Dashboard
This dashboard provides an optimized overview of all relevant information about your Bitcoin holdings. It is designed to help both newcomers and long-term Bitcoin investors gain a balanced overview of their transactions and compare them with current market values.
You can enter your own transactions (including past transactions). The dashboard analyzes this data and visualizes your BTC holdings based on current market values.

### The following features are available:
- Automated market value queries
- Transaction tracking (also possible in different currencies)
- Wallet management and overview
- Analysis charts
- Solominer monitoring (AxeOS)
- Report generation (account overview)
- 24-hour price history

### Languages:

- Automatic backup Reporting Features
